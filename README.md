# Expense Tracker

A simple and user-friendly Python Expense Tracker application that allows you to record daily expenses, view them, calculate total spending, and visualize them using charts.

This project is ideal for beginners learning **CSV handling**, **file operations**, **user input**, **dictionaries**, and **matplotlib**.

* Add new expenses with category and amount
* Store expenses in a CSV file
* View all recorded expenses
* Calculate the total spending
* Plot bar chart of expenses by category using matplotlib
* Auto-create CSV file if it doesn't exist
